# romannumeralcalculator
Roman Calculator
Problem description
King Caesar would like us to create the calculator program, which can add a Roman numeral in order to save our manual effort.
 
For example
o   “XX” + “II” = “XXII”
o   “II” + “II” = “IV”
o   “XIV” + “LX” = “LXXIV”
o   “V” + “V” = “X”
 
•    Input file format
 
XX+II
II+II
XIV+LX
V+V
 
•    Output format
 
XXII
IV
LXXIV
X
